Project 1: Logistic Regression basics

Description

This project involves a scratch implementation of logistic regression, accompanied by data analysis and a comparison with existing methods in scikit-learn. The primary application of this method is in modeling the classification of cancer tumors, specifically to determine whether they are benign or malignant. Additionally, the analysis explores strategies for creating a lighter model using correlation analysis and Principal Component Analysis (PCA).

View on GitHub

Objective

  • Apply gradient descent optimization to minimize the error.
  • Leverage gradient descent to estimate optimal model parameters with bias.
  • Compare the accuracy between the custom model and the Sklearn implementation.
  • Conduct a correlation analysis to identify relationships between variables in the data.
  • Perform Principal Component Analysis (PCA) to reduce dimensionality of the data.

Results

Running for 10,000 iterations, achieves an error rate of about 0.1 and begins to converge. To find optimal parameters, we take the final parameters and restart the iteration process until we reach convergence or achieve a minimal error.

The gradient descent method achieves a better porcentage of accuracy of 98.8% respect Sklearn with 98.2%. Both scores are excellent, indicating that almost every type of cancer is correctly classified.

Through correlation analysis, we identified several features that could serve as strong candidates for the model. However, when evaluating accuracy, we observed a decrease of about 5%, bringing the overall accuracy to 94%. While this is still a commendable performance, the context—given that it involves human lives—requires a higher degree of certainty. Therefore, it is crucial to ensure the model's reliability.

In this analysis, six components effectively explain the data, which is why it is selected only six features to develop the model. Principal Components One and Two exhibit a clear linear relationship, leading to more compelling results. The model achieved an impressive accuracy of 0.97, which is nearly equivalent to the performance with all features. Consequently, the PCA analysis significantly reduced training time and model complexity.

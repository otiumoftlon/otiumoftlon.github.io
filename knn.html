Project 1: K-nearest Neighbor Classifier

Description

This project involves a scratch implementation of k-nearest neighbor classifier, accompanied by small data analysis and a comparison with existing method in scikit-learn. I applied this method over the wine classifier dataset provided by sklearn. Additionally, the analysis explores the grades of accuracy for different k.

View on GitHub

Objective

  • Apply the k-nearest neighbor algorithm using Euclidean and Manhattan distances.
  • Utilize the k-nearest neighbor algorithm for classification purposes.
  • Conduct an exploratory data analysis (EDA) on the dataset.
  • Compare the custom implementation with Sklearn's implementation using different values of K.

Results

As a result, I focused on exploring the relationships between features to identify which classifier might perform best. Using Seaborn's pairplot to generate pairwise scatter plots for each feature combination, I was able to visualize potential patterns, correlations, and class separability. This analysis provided valuable insights that helped guide the selection of a suitable classifier in this case a knn classifier.

The results show that for most values of K, my method achieves similar accuracy to the sklearn implementation. However, for K=2, my method slightly outperforms sklearn, while for K=8, 10 and 12, sklearn performs better, particularly at K=8 where it achieves the highest accuracy of 0.98.

Overall, the model yields an average accuracy of 0.964, which is a strong result for this task. Based on these findings, I recommend using a lower value of K, such as K=4, to keep the algorithm efficient without risking overfitting. This strikes a good balance between performance and generalization for future datasets.

Project:1 Linear Regression Basics

Description

This Python code generates a function that applies linear regression using gradient descent based on absolute error. This method is utilized to model an eCommerce dataset, predicting the yearly amount spent by users. Additionally, it incorporates Principal Component Analysis (PCA) and correlation analysis to identify the best features and gain insights into the data.

You can view the code on GitHub.

Objective

  • Apply gradient descent optimization to minimize the error.
  • Leverage gradient descent to estimate optimal model parameters.
  • Compare the Mean Absolute Error (MAE) between the custom model and the Sklearn implementation.
  • Conduct a correlation analysis to identify relationships between variables in the data.
  • Perform Principal Component Analysis (PCA) to reduce dimensionality of the data.

Results

The gradient descent method achieves a good result only variying respect Sklearn by 0.1%. The regression task shows also a an error of 17.35 in term of houndred of dollars is a good parameter nevertheless is better to try another approaches to reduce that error.

The correlation analysis reveals that the Length of Membership and Time on Website are the features most strongly correlated with Yearly Amount Spent. Interestingly, the Time on App shows no significant correlation. This insight can be omitted from the model and provides valuable information for future website improvements. By making the website more user-friendly, we can encourage increased customer purchases.

Unfortunately, performing PCA did not yield significant benefits. Given this, and considering the relatively small number of input features, it is more practical to retain the original features.
